# App

## How to start

- Install node (v22.20.0)
- Install yarn
- Get repository
- Open the project directory
- Install dependencies `yarn install`
- Start developer mode `yarn dev`
- Available pages: `/` and `/prices`

## Tools used:

- Nuxt 4
- Tailwind
- Pinia
- Drizzle Orm
- SQLLite
- TypeScript
- ESLint
- Prettier
